# Tools Info
Many tools have a succint description in tools.info. You can list them all with:
```
 cat MCCE4-Alpha/MCCE_bin/tools.info
```

For more information about a given tool, check the wiki, or use the "-h" flag, e.g.:
```
step3.py -h
```
